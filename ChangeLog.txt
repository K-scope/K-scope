CHANGELOG
=========

(2012/3/15) Version 0.1.0 is released.
- A new parser, fonrt end of XcalableMP compiler is supported.
- New features are implemented to enahance usabilities.

(2013/3/22) Version 0.2.0 is released.
- Added an imtermediate code building via K-scope on local computer
- Added a visualization for profiling data for the K computer
- Added a calculation for required byte/FLOP

(2013/4/12) Version 0.2.1 is released.
- Added an alert feature when finding procedures with the same name in structure analysis
- Enhanced a viewer for additional information
- Fixed bundle resources properties to refactor
- Fixed a folder name: "project.ksx" to "Kscope_project.ksx"
- Fixed a file name: "settings" to "Kscope_settings"

(2013/5/27) Version 0.2.2 is released.
- Added a file filter in dialog of open project
- Changed a feature of text output, csv-form to tree-form
- Fixed a file tree construction to improve performance
- Fixed a bug: cost ruler for profiling data is not repainting by new event
- Fixed a bug: case sensitive is not aware in the tree-view
- Fixed a bug: exception arises when calling an external tool in Linux environment
- Fixed message resource files to increase maintainability
- Fixed an analysis-view interface
- Fixed an implementation for JDK7 instead of JDK6 which is end of support in the K-scope.

(2013/5/31) Version 0.2.3 is released.
- Added and removed icon files as minor changes
- Fixed "build.xml" to specify UTF-8 encoding
- Removed some properties to refactor
- Fixed bugs for Mac OS X envrironment as follows:
 * A background color of unselected node in the tree-view is not repainted.
 * The "Class Cast Exception" arises in the variable access settings.
 * The file selection was not selectable in new project dialog. (only Mountain Lion and Java 1.7)

(2013/10/31) Version 0.2.4 is released. 
This release is collaboration version between AICS HPC Usability Research Team and AICS Software Development Team.
- Bundled new library, SSHconnect.jar into the binary package
- Added a building feature for intermediate code on server via SSH port in full project mode
- Fixed a main class name: "Go" to "Kscope"
- Fixed an archive format: zip to tgz
- Fixed small changes in the interfaces
- Fixed a few typos in messages and captions

(2013/12/27) Version 0.2.5 is released.
- Hide passwords in server settings table
- Version upgrade for SSHconnect (1.03 to 1.18) to support front-end of the K computer
